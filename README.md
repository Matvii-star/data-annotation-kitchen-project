# data-annotation-kitchen-project
## Overview

This project demonstrates my practical experience with image annotation using CVAT as part of my preparation for Data Annotation and Data Tagger positions.

The goal was to gain hands-on experience with common computer vision annotation tasks used for training machine learning models.

## Project Scope

I annotated a collection of kitchen images and practiced multiple annotation techniques commonly used in computer vision datasets.

## Annotation Types

### Bounding Box Annotation

Object detection using rectangular bounding boxes.

Annotated classes:

* Cabinet
* Sink
* Refrigerator
* Stove
* Countertop

### Polygon Segmentation

Precise object outlining using polygon shapes for more accurate object boundaries.

### Key Point Annotation

Key point placement on selected object features to understand landmark annotation workflows.
